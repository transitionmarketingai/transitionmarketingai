   ```
   NEXT_PUBLIC_SUPABASE_URL
   NEXT_PUBLIC_SUPABASE_ANON_KEY
   SUPABASE_SERVICE_ROLE_KEY
   ```

2. **Twilio (WhatsApp):**
   ```
   TWILIO_ACCOUNT_SID
   TWILIO_AUTH_TOKEN
   TWILIO_WHATSAPP_NUMBER
   ```

3. **Resend (Email):**
   ```
   RESEND_API_KEY
   ```

4. **OpenAI (AI Scoring):**
   ```
   OPENAI_API_KEY
   ```

5. **Facebook (Meta Ads):**
   ```
   FACEBOOK_APP_ID
   FACEBOOK_APP_SECRET
   FACEBOOK_ACCESS_TOKEN
   FACEBOOK_VERIFY_TOKEN
   ```
